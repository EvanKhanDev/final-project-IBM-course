# Simple Interest Calculator

This repository contains a Bash script that calculates **simple interest** based on user input.

## Inputs
- principal
- rate of interest
- time period

## Formula
Simple Interest = (principal * rate of interest * time period) / 100

## How to run

simple-interest.sh
